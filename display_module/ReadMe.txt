In this folder we will use st7735 module.
all the codes can be changed for other modules as well as per requirement.
Most of the codes are begener friendly some are little bit tricky and some are to complex to understand.
For flashing the code use aurdino cloud sservice foor the best support.
For flashing the code in you esp32 devkit V1
1st) Make a account on aurdino cloud.
2nd) Download the desireable file.
3rd) Upload it on aurdino cloud.
4th) Compile it and select esp32 devkit v1
5th) Click on the upload button.
Note that the require agent and the drivers must be pre-installed so the code can be flashed easily.
